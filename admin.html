图片生成API

在「API Provider」下拉框中选择 Nano Banana（OpenAI 兼容） 或 Nano Banana（Gemini 原生） 后，会展开 Base URL、模型等配置项。

本系统已实现的路由（相对当前站点根路径）
  • POST /api/generate/image — 用户出图（需前台登录 Cookie）。prompt 必填；多参考图传 reference_urls（HTTPS URL 数组，与官方 Gemini parts 中多个 imageUrl 一致，最多 5 张，超出自动截断）；亦可 reference_url / image_url 单张；另有 aspect_ratio、resolution、gemini_model 等。
  • GET /api/admin/nano-banana/usage — 代理 Nano Banana 用量查询（需管理员登录）。Query：days、limit，例如 ?days=7&limit=50。
上游 Nano Banana（由上方 Base URL 拼接，默认 https://banana.aigenmedia.art）
  • OpenAI 兼容：POST {base}/api/v1/chat/completions
  • Gemini 原生：POST {base}/api/gemini/v1beta/models/{model}:generateContent
  • 用量：GET {base}/api/usage

视频生成API

对接 拍我AI 开放平台（PixVerse OpenAPI v2）。流程说明见 API 工作流程。请求头 API-KEY、Ai-trace-id（本站每次请求自动生成）；图生视频需公网 HTTPS 可访问的参考图 URL（本站上传请配 PUBLIC_SITE_URL）。图生视频在本站按下方「图生视频 quality」单次提交（如 1080p），不再做预览与二次超清两步。

要「成片里自带声音」？ 这是拍我在出片时一起合成的对白/环境音，写进返回的 MP4，不需要您上传任何音频文件。请在下方勾选「生成视频自带声音」并保存；依赖接口字段 sound_effect_switch 等。model c1 与 v6（及 v6.*）当前网关不支持该字段，本站请求中不会附带；v3.5 / v5 等以实际接口为准。

超级编导等图生视频一次请求即使用该档位，不再区分「预览 / 二次超清」。

若接口提示 motion_mode:fast is only supported for models v3.5, v4, and v4.5：请选 normal，模型用 c1（C1）。本站请求时会自动把不兼容组合降为 normal。

勾选后请求里会带 sound_effect_switch: true，让上游在生成 MP4 时写入音轨（对白/环境音等），不是让您上传 wav/mp3。model c1 与 v6（及 v6.*）本站不会附带该字段；其它模型以接口为准。下方「声音描述」仅为文字提示。新建默认不勾选（与 c1 默认搭配）。
