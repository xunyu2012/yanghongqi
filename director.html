分镜要求 · 支持格式 · 站外大模型提示词（点击展开/复制）

每条分镜须满足（便于后续成图/成片）

  • 画面：景别 + 机位或运镜 + 具体画面 + 节奏/情绪；竖屏短剧，中文，单条建议不少于 35 字，避免「很美」「很紧张」等空泛词。
  • 人物：本镜出场者「姓名（身份）」；无人物写 无；多人用顿号、逗号均可。
  • 顺序：镜号按剧情时间递增，勿跳号、勿合并多镜为一行。
  • 对白请用直角引号「」，勿使用未转义的英文半角 "。

格式一（推荐，解析最稳）

每镜一块，首尾用分隔行（必须完全一致、单独一行）：

<<<XUNYU_SHOT>>>
镜号: 1
画面: 中景，主角站在村口老槐树下，侧逆光，表情凝重；手持信纸微微颤抖。
人物: 张三（村民）
<<<XUNYU_SHOT>>>
镜号: 2
画面: 过肩镜头越张三肩头拍向村长，村长皱眉摇头，背景虚化为祠堂轮廓。
人物: 张三（村民）、李四（村长）

键名可用：画面/描述/镜头、人物 / 出场人物 / 演员 / 角色、可选 运镜 / 运镜方式（基础稳定六项之一）、空间与氛围运镜 / cameraMoveSpace（第二组五项之一）、高级情绪运镜 / cameraMoveEmotion（第三组五项之一）、视角与叙事运镜 / cameraMoveNarrative（第四组四项之一）、长镜头与转场运镜 / cameraMoveLongTrans（第五组四项之一）、航拍专用运镜 / cameraMoveAerial（第六组四项之一），否则按不选处理。导入后会按姓名与「角色」页素材名匹配并填入主体参考。第一块前可有剧本说明文字，会被忽略。

格式二（Markdown 分节）

### 镜号 01
- **画面**：……
- **人物**：……

### 镜号 02
- **画面**：……

三级标题须含「镜号」或「分镜」加数字；每条下用列表写画面与人物；可选 - 运镜：、- 空间与氛围运镜：、- 高级情绪运镜：、- 视角与叙事运镜：、- 长镜头与转场运镜：、- 航拍专用运镜：（取值须与各下拉分组内选项完全一致）。

格式三（JSON 数组）

[
  { "description": "画面文字……", "characters": "人物……", "cameraMove": "跟镜头", "cameraMoveSpace": "推镜头", "cameraMoveEmotion": "手持晃动", "cameraMoveNarrative": "过肩镜头", "cameraMoveLongTrans": "长镜头", "cameraMoveAerial": "航拍上升" },
  { "description": "……", "characters": "无" }
]

正文里可夹其它说明；解析时会取第一个 [ 到最后一个 ] 之间的 JSON。

复制给站外大模型的系统提示词（任选其一模型粘贴为「系统」或首条用户消息）
